2020U Assignment 1
==================

Author(s):
Luke Tran

About:
A java project that simulates a Bayesian spam detector.

Download
1. Download can be found at https://github.com/Luke-Tran/csci2020u_assignment1

2. Clone the project or download the zip file and extract it

Usage
1. Open the folder where the project has been cloned or extracted

2. Navigate the folders: out -> artifacts -> Assignment1_jar

3. Run "Assignment1.jar"

4. Open the folder that contains the train and test folders.

5. Wait until the program finishes processing the files.

6. A table of the files and their spam probabilities will open.
